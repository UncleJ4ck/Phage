# Contributing

Phage is a personal security-research fork. Issues and pull requests are welcome
at https://github.com/j4kuuu/phage.

- Keep changes focused and tested. Run `python -m pytest tests/` before opening a PR.
- The evolutionary engine lives under `src/phage/evo`; see `docs/EVO.md`.
- Match the existing style: `ruff format` and `ruff check` must pass.
